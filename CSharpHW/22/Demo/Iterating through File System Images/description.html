Introduction

How can we use computer vision in a practical way and how successful is it? In this series of samples we will create a computer vision system which detects and identifies faces within a collection of images. This first sample will show you how to iterate and display the sample images that we will be using.

Building the Sample

The sample is written in C# within Visual Studio 2010 on a Windows Server 2012 machine. It will run on all appropriately configured Window XP computers and better.

Description

To detect and identify images we need a collection of images to start with. If you do not have a reasonable collection of test images then download and run my Web Image Crawler and Downloader from here which will let you quickly generate a nice collection of images.

Next we need to iterate through our images.There are many ways to accomplish this but for this sample we have chosen the most simplest way possible.

As the whole system can be run automagically we do not need a user interface, but to illustrate what is happening we have a very basic interface which in this sample will simply show all the images it can in a user selected folder. If the folder and contents are massive it can be stopped at any point.

All we really do here is check to see if the program is starting or stopping, If it is starting then load the files from the specified folder selected from the FBD (FolderBrowserDialog). We then get all the files in that folder and its sub folders. We compare the extension to a list of predefined image extensions and try to display each image in the PictureBox (pbImages).

 

If the Start/Stop button is pressed again - the foreach loop is ended and the program stops.
