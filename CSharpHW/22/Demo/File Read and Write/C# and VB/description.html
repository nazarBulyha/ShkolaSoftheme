Introduction

Console applications are sometimes all that are necessary for our applications. But, Console applications are just not so well understood. In this tutorial we show you how to Ask questions, Get the Answers and save the results in a nice way.

Building

This app was written in Visual studio 2010 on a Windows Server 2012RC x64. The solution will load in any appropriate Visual Studio and on any appropriately configured Windows XP or better machine.

Description

Within the forums an individual was having difficulty with their console app haveing an infinite loop and they could not see where the problem was. I have been coding for 30 years and found it quicker to write their app than debug it because of the design of the application. It was the design of the application that was causing them the problems.

This solution shows how to design your application to ask questions, collect the answers, output all the replies and also save and load the responses.

 As you can see the program is split into 5 sections (methods). We have the Main() entry point for the program, a method which Asks the questions and collects the answers. We then have a method which saves the results to a file, and another which loads the results from the file. Finally we have a helper method which waits for the user to press a key before continuing.

Within main, we loop until the user no longer wants to enter data. It contains the list of questions to ask. In a real world applications the questions would be read in from a file or database; however, I have hard coded them for this simple example.
